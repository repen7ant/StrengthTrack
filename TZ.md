# StrengthTrack — система планирования силовых тренировок для опытных натуральных атлетов

## 1. Название и концепция

**StrengthTrack** — это веб-приложение для планирования силовых тренировок на основе принципов **RPE / RIR** и фиксированной **4-недельной периодизации**.

### Проблема, которую решает продукт

Опытные натуральные атлеты часто планируют тренировочные циклы вручную, что приводит к:

- неточным расчётам рабочих весов;
- отсутствию единой логики прогрессии;
- сложности отслеживания роста силы во времени.

StrengthTrack автоматизирует расчёт нагрузок для **трёх основных упражнений** (присед, жим лёжа, становая тяга) и предоставляет понятную систему мезоциклов, оставляя свободу в подборе вспомогательных упражнений.

## 2. Целевая аудитория и роли

### Целевая аудитория

- Опытные атлеты (intermediate / advanced)

- Пользователи, знакомые с RPE / RIR и принципами периодизации

| Роль                            | Описание возможностей                                                             |

## 3. Схема данных (Сущности)

### 1. UserProfile (Профиль пользователя)

**Описание:** Расширение стандартного пользователя, хранящее тренировочные данные.

**Ключевые поля:**

- `user` — связь с Django User
- `created_at` — дата регистрации
- `last_mesocycle_start` — дата последнего мезоцикла

---

### 2. Exercise (Упражнение)

**Описание:** Справочник упражнений.

**Ключевые поля:**

- `name` — название упражнения
- `is_main` — флаг (присед / жим / тяга)
- `muscle_group` — целевая группа мышц

---

### 3. BestSet (Лучший подход)

**Описание:** Лучший зафиксированный подход пользователя в упражнении.

**Ключевые поля:**

- `user` — владелец результата
- `exercise` — упражнение
- `weight` — вес
- `reps` — количество повторений
- `estimated_1rm` — расчётный 1RM
- `date` — дата фиксации

---

### 4. Mesocycle (Мезоцикл)

**Описание:** 4-недельный тренировочный цикл для одного упражнения.

**Ключевые поля:**

- `user`
- `exercise`
- `start_date`
- `week` — номер недели (1–4)
- `rpe` — целевой RPE недели
- `rir` — целевой RIR недели
- `target_weight` — расчётный рабочий вес
- `target_reps_range` — примерный диапазон повторений

---

## 4. Основные сценарии пользователя (User Stories)

### 1. Ввод лучшего подхода

**Как пользователь**, я ввожу лучший подход (вес и повторения) для любого упражнения,  
**чтобы** система рассчитала мой estimated 1RM и сохранила его в профиле.

---

### 2. Генерация мезоцикла

**Как пользователь**, я выбираю дату начала мезоцикла,  
**чтобы** получить 4-недельный план для приседа, жима лёжа и становой тяги.

Если для любого из трёх основных упражнений не заполнен BestSet — система выводит ошибку.

---

### 3. Просмотр информации о мезоцикле

**Как пользователь**, я просматриваю:

- рабочие веса и примерные повторения;
- фиксированные RPE и RIR для каждой недели;
- напоминание о необходимости **10–20 подходов на мышечную группу в неделю**.

---

### 4. Завершение цикла и повторное планирование

**Как пользователь**, после завершения мезоцикла я:

1. обновляю лучшие подходы,
2. задаю новую дату старта,
3. получаю следующий мезоцикл.

---

### 5. Анализ прогресса

**Как пользователь**, я хочу видеть график изменения estimated 1RM по датам  
**чтобы** отслеживать рост силы в приседе, жиме и тяге.

---

## 5. Технический стек и интеграции

| Компонент         | Технологии                            |
| ----------------- | ------------------------------------- |
| Backend           | Python, Django                        |
| База данных       | PostgreSQL / SQLite                   |
| Frontend          | Django Templates, Bootstrap           |
| Расчёты           | Формулы расчёта 1RM (Epley / Brzycki) |
| Графики           | Chart.js или Plotly                   |
| Аутентификация    | Django Auth                           |
| Администрирование | Django Admin                          |

---

**Итог:** StrengthTrack — это инструмент строгой, но гибкой периодизации для опытных атлетов, который автоматизирует расчёты, упрощает планирование и делает прогресс наглядным.
